
Zero Starvation 🌍🍱

Zero Starvation is a web application built with the vision of reducing food waste and helping fight hunger by connecting food donors with people and organizations in need.
The platform aims to create a simple, accessible, and community-driven ecosystem where surplus food can be shared instead of wasted.

⚠️ Project Status: This project is currently under development and is not yet complete. Features, UI, and backend functionality are still being actively worked on.

🎯 Project Goal

Millions of tons of food are wasted every day while many people struggle with hunger.
Zero Starvation aims to bridge this gap by building a platform where excess food can be donated efficiently to people who need it.

The goal is to:

Reduce food wastage
Support underprivileged communities
Encourage social contribution
Build a sustainable food-sharing ecosystem

⚙️ Installation & Setup

Clone the repository:

git clone https://github.com/your-username/zero-starvation.git

Navigate to the project folder:

cd zero-starvation

Install dependencies:

npm install

Create a .env file and add your Supabase credentials:

VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key

Start the development server:

npm run dev
